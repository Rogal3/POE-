# POE-Flask-macro
오토핫키 기반 매크로

* 오토 핫키를 설치한다.
* 니 직박구리 폴더에서 우클릭 - 새로만들기 - autohotkey script 를 클릭한다.
* 새 autohotkey를 우클릭하고 edit을 클릭한다.
* https://github.com/Rogal3/POE-Flask-macro/blob/master/2to12345.ahk 코드를 복붙하고 저장한다.
* 저장한 파일을 우클릭하고 관리자 권한 실행한다.
